###FreshCart E-commerce Platform Built with React.js

##Description

FreshCart is a full-featured e-commerce application built using React.js. It allows users to:

Create accounts and log in securely
Browse and search for products
Add items to their cart for purchase
Save desired products in a Wishlist for later consideration
Choose their preferred payment method: cash on delivery or secure online payment integration with Stripe
Features

User Authentication & Management: Secure account creation and login functionality.
Product Management: Product browsing, cart functionality, and Wishlist management.
Flexible Payment Processing: Cash on delivery and Stripe integration.
Data Management: Efficient data fetching, sending, deletion, and updates using Axios.
Enhanced User Experience: Improved UX through data caching (react-query) and pagination for smooth product browsing.
SEO Optimization: Implemented react-helmet for unique meta titles and descriptions per component, improving search engine visibility.
Error Handling: Included a "Not Found" page for user-friendly navigation.
Technologies Used

###Frontend Development: React.js, React Router, Formik, Yup, react-helmet, react-query
###API Integration: Axios
###Styling: Sass, Bootstrap, Font Awesome
